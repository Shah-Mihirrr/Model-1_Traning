Java Script Math Functions

JavaScript provides a built-in Math object that contains various mathematical functions and constants. Here are some commonly used Math functions:

  • Math.abs(x): Returns the absolute value of a number.
  • Math.ceil(x): Rounds a number up to the nearest integer.
  • Math.floor(x): Rounds a number down to the nearest integer.
  • Math.round(x): Rounds a number to the nearest integer.
  • Math.max(a, b, ...): Returns the largest of zero or more numbers.
  • Math.min(a, b, ...): Returns the smallest of zero or more numbers.
  • Math.pow(base, exponent): Returns the base raised to the power of exponent.
  • Math.sqrt(x): Returns the square root of a number.
  • Math.random(): Returns a random number between 0 (inclusive) and 1 (exclusive).

You can use these functions in your JavaScript code to perform various mathematical operations. For example:

// Example usage of Math functions
let num = -5;
console.log(Math.abs(num)); // Output: 5
console.log(Math.ceil(4.3)); // Output: 5
console.log(Math.floor(4.7)); // Output: 4
console.log(Math.round(4.5)); // Output: 5
console.log(Math.max(2, 5, 1, 9)); // Output: 9
console.log(Math.min(2, 5, 1, 9)); // Output: 1
console.log(Math.pow(2, 3)); // Output: 8
console.log(Math.sqrt(16)); // Output: 4
console.log(Math.random()); // Output: Random number between 0 and 1

These functions can be very useful for performing calculations, generating random numbers, and manipulating numerical data in your JavaScript applications.
